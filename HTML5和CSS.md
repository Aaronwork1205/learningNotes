# HTML5 和 CSS笔记



### 注释和类型



- 文档声明

  - 用于声明浏览器当前网页版本, html5的文档声明, 写于html文档开头，不区分大小写。

    ```html
    <!doctype html>
    <!Doctype heml>
    <!doctype HTML>
    ```

- 注释写法

  ```html
  <!-- /注释内容/ --!>
  ```

- meta标签

  - 用于标注网页的字符集，避免乱码的问题， 写于head标签内

  - 常用字符集 utf-8万国码

    ~~~ html
    <meta charset="utf-8">
    ~~~
